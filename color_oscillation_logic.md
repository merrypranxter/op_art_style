
# Color Oscillation Logic

Colour animation should occur at boundaries, thresholds, specific objects, or selected zones. Avoid global rainbow cycling.

**Modes:**
- **Force-Color Pulse:** Intensify a single accent hue periodically.
- **Threshold Charge Sweep:** Pulse colour along lip rings or portal edges.
- **Blacklight Glow Cycling:** Oscillate intensity of UV hues within a palette family.
- **Prismatic Edge Flicker:** Oscillate spectral fringes at active edges.
- **Contamination Bloom Drift:** Allow localised material zones to cycle through iridescent or acid hues.
- **Route-Light Sequencing:** Sequentially illuminate path markers.

Always retain palette identity while animating. Use oscillations to emphasise structure: centre, threshold, or object rather than global colour.
