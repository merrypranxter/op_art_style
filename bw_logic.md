
# Black-and-White Logic

Black and white are the structural bones of Retinal Surrealism. Pure value contrast clarifies rhythm, intensifies tension, and exposes spacing errors.

**Roles:**
- Provide the scaffolding for pattern systems.
- Generate vibration and figure–ground instability.
- Anchor colour accents.
- Ensure images survive in monochrome form.

**Modes:**
- **Pure Severe Monochrome:** Only black and white. Used for classic Op branches and testing pattern strength.
- **Soft Value Modulated Monochrome:** Black, white, and greys used to create false depth and atmospheric transitions.
- **Monochrome + Minimal Accent:** Black, white, and one accent hue used sparingly to highlight features or objects.

**Best practices:** Build patterns in black‑and‑white first. Use black as weight or danger; white as release or flash. Avoid midtone mush unless specifically desired for depth; crisp value jumps create stronger optical force.
