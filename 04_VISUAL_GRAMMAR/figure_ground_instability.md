
# Figure-Ground Instability

Figure–ground instability occurs when the viewer cannot decide what is object and what is background. It produces tension and motion within a static image.

**Types:**
- Contrast reversal: alternating bands flip between figure and ground.
- Pattern merge: objects share the same pattern as the field.
- Depth instability: shapes can be both bulge and recess.
- Silhouette dissolve: figure edges fade into the field.
- Multi‑read: images that support multiple reads at different distances.

**Uses:**
- Make bodies merge with environment.
- Hide eyes or symbols in pattern fields.
- Create hidden forms that appear at certain distances.
- Generate false motion as the eye flips between interpretations.

**Best practices:** Use figure–ground instability intentionally. Preserve some anchor points to prevent total confusion. Merge zones should be motivated by pattern logic, not randomly smeared.
