
# Prompt Templates

This file provides high‑level templates for constructing Retinal Surrealism prompts. Each template describes composition type, optical mechanism, pattern mechanics, spatial logic, figure/object/world integration, colour logic, material logic, and mood.

**Template Structure:**
`[composition] built from [optical system], using [pattern mechanics] to create [perceptual effect]; [figure/object/world integration]; [colour logic]; [material logic]; [mood/branch identity].`

**Examples:**
- *Pure Optical Field:* “A severe black‑and‑white optical field built from tightly packed concentric targets and radial compression, creating false tunnel depth; crisp edges, strong contrast hierarchy, no decorative clutter.”
- *Op Landscape:* “A surreal optical landscape where rolling dunes are generated through black‑and‑white contour bands, with a target‑stone path leading toward a spoke‑burst horizon portal; the world behaves like patterned dream geography; blacklight accents intensify thresholds.”
- *Op Bodysuit Figure:* “An iconic figure wearing a high‑contrast contour bodysuit built from zebra waves, pattern tightening at waist and hips; environment shares the same pattern logic; black‑and‑white structure with hot pink accent on hair and lips; glamorous and confrontational.”
- *Glam Portrait:* “A confrontational portrait where the face merges with a contour field and halo target, with prismatic edge charge around eyes and lips; hot pink hair acts as force colour; the portrait feels like a staged perceptual confrontation.”

Use these templates to begin prompts, adjusting details for branch, pattern family, colour, and mood.
